# akakce.com Automation Project

## About the Project

The main purpose of this project is to automate and validate core user flows on the Akakce.com website, such as:

Account creation
Login & logout
Account verification
Favorites management
Contact form validation
The framework is structured to ensure maintainability, reusability, and clear reporting using clean coding practices.
